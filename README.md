# 🛢️ IPR-TPR Model for Oil Well Performance

This project simulates the performance of an oil production well by modeling both **Inflow Performance Relationship (IPR)** and **Tubing Performance Relationship (TPR)**. The goal is to determine the well's production rate by identifying the operating point — the intersection of inflow and outflow curves.

Built using **Python** and **Google Colab**, this notebook provides a simple visualization of well behavior under different flowing conditions.

---

## 📘 What This Project Covers

- ✅ Calculation of flowrate using **Vogel’s IPR equation**
- ✅ Generation of a mock TPR curve (representing tubing pressure losses)
- ✅ Plotting both IPR and TPR to find the **operating point**
- ✅ Fully functional in **Google Colab**, no external data files needed
